SmartBite is an AI-powered cooking assistant built with **Chainlit**, **YOLO11**, and **Cohere LLM**.  
The system detects ingredients from fridge images and suggests creative recipes based on the detected items or user descriptions.

---

##  Features

-  **Ingredient detection** using YOLO11  
-  **AI recipe generation** via Cohere  
-  **Smart shopping cart** with add/view/clear features  
-  **Interactive chat interface** using Chainlit  
-  **Automatic recipe images** from Unsplash  
-  Supports **text** and **image** inputs

---

| Component | Description |
|----------|-------------|
| **Python** | Main programming language |
| **Chainlit** | Conversational UI |
